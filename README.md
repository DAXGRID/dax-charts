# DAX Helm Charts

Helm charts for DAX organization using [Helm](https://github.com/helm/helm) for Kubernetes.

## TL;DR

```bash
helm repo add dax https://daxgrid.github.io/charts/
helm repo update
helm install my-release dax/<chart>
```
